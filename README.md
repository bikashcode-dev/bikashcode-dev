## 🚀 Featured Project

### 📧 AI Email Reply — Gmail Chrome Extension

> A Chrome Extension that injects an **AI Reply button** directly inside Gmail.  
> One click generates a professional, human-like reply using **Gemini AI** with **Groq** as fallback —  
> all powered by a **Spring Boot** REST API backend.

**What it does:**

| Feature | Details |
|:---|:---|
| 🔌 Gmail Integration | AI Reply button injected inside Gmail's compose toolbar |
| 🎭 Smart Tone | 6 tone options: Professional, Friendly, Formal, Casual, Polite, Confident + **Auto Detect** |
| 🤖 Dual AI Engine | Gemini AI as primary, Groq as fallback — always available |
| ✍️ Human-like Output | No robotic-sounding text — replies feel natural |
| 🔁 Full Thread Support | Works on both new compose and reply threads |
