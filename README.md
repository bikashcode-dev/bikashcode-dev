## 🚀 Featured Project

<div align="center">

### 📧 AI Email Reply — Gmail Chrome Extension

</div>

> A **Chrome Extension** that injects an AI Reply button directly inside Gmail.
> One click generates a professional, human-like reply using **Gemini AI** (Groq as fallback) —
> powered by a **Spring Boot** REST API backend.

```
✅  AI Reply button injected in Gmail compose toolbar
✅  6 tone options — Professional, Friendly, Formal, Casual, Polite, Confident
✅  Auto tone detection based on email context
✅  Gemini AI primary + Groq fallback — always available
✅  Human-like replies — zero robotic-sounding text
✅  Works on both compose window and reply thread
```
